# Binary Sentiment Classification

This project focuses on classifying sentiments from various text sources using a binary classification approach. The main goal is to accurately determine whether the sentiment expressed in a given text is positive or negative.

## Pipeline Overview

![Pipeline](binary_sentiment_classification_pipeline.png)

## Data Sources
The dataset used for training and testing includes:
- Amazon reviews
- IMDB reviews
- Yelp reviews

## Methodology
The project employs various natural language processing techniques and machine learning algorithms to achieve high accuracy in sentiment classification.

## Results
The results of the classification are documented in the report. Please refer to the report for detailed insights and performance metrics.

## Installation
To set up the project, clone the repository and install the required dependencies.

## Usage
Run the scripts in the `src` directory to execute the sentiment classification tasks.
